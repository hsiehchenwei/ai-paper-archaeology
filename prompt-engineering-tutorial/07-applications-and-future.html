Chapter 07

應用與未來

從 2021 年的預言，看 2026 年的現實。Prompt Engineering 如何從一種技巧，演變成構建 AI 應用的基礎設施？

在這篇論文發表的 2021 年，Prompting 還只是一種「新興的 NLP 範式」。作者列舉了許多潛在的應用場景，如今大多已成為我們日常使用 AI 的標準方式。

1. 知識探勘 (Knowledge Probing)

"Factual probing (a.k.a. fact retrieval) is one of the earliest scenarios... knowledge is retrieved by transforming the original input into a cloze prompt... e.g. 'iPod Touch is produced by [Z].'"

把模型當知識庫

論文最早的應用之一是 LAMA (Language Model Analysis)，測試模型到底記住了多少事實。

現在，這已經演變成了 RAG (Retrieval-Augmented Generation) 的反面——直接利用模型內在知識。雖然我們現在更傾向於用外部知識庫來減少幻覺，但「模型內在知識」依然是推理的基礎。

2. 元應用 (Meta-Applications)

這部分非常有趣，論文預見了 Prompt 不僅能解決 NLP 任務，還能用來「改善模型本身」。

"Dataset Construction: ...suppose we have an unlabeled dataset... we can use the following template... 'Write two sentences that mean the same thing. [X][Z]'"

合成數據生成 (Synthetic Data)

論文提到用 Prompt 來生成訓練數據。這在當時是為了省標註錢，現在則是因為「人類數據不夠用了」。

現代使用對照

GPT-4 訓練了 Llama 3。

現在開源模型的強大，很大程度上歸功於使用了 GPT-4 生成的高質量合成數據進行微調。這篇論文在 2021 年就指出了這條路徑：用強模型的 Prompt 能力來生產數據，訓練弱模型。

"Debiasing: ...if the probability of 'Yes' is greater than 'No', then we would assume the given text contains violence... These two probability distributions... can be combined to suppress the undesired attribute."

AI 自我審查 (Self-Correction)

論文提出可以用 Prompt 讓模型自我診斷偏見。

現代使用對照

Constitutional AI (憲法 AI)。

Anthropic 的 Claude 模型使用的 Constitutional AI 技術，核心思想就是用一組 Prompt (憲法) 讓模型自我批評、自我修正。這正是論文中 Debiasing 思想的規模化實踐。

2021 -> 2026: 預言成真

這篇論文的偉大之處，在於它將 Prompt 從一種「小技巧」(Trick) 提升到了「範式」(Paradigm) 的高度。

它預言了：

  • 我們不再微調模型，而是微調 Prompt (In-context Learning)。
  • 我們會用 Prompt 來連接不同的任務 (Agent)。
  • Prompt 會成為一種新的程式語言 (Prompt Engineering)。
現在，這一切都已發生。我們正生活在第四範式的世界裡。
